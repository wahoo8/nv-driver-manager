# Security Policy

## Supported Versions

During development only the latest development branch is supported.

After Version 1.0 is released, security fixes will be backported when practical.

## Reporting a Vulnerability

Please do not publish security vulnerabilities publicly.

Instead, contact the maintainer privately.

Include:

- Debian version
- Kernel version
- NVIDIA driver version
- Steps to reproduce
- Relevant log files

Security issues involving privilege escalation, Secure Boot,
PKEXEC, DKMS or downloaded installers should be reported immediately.

## Scope

This project downloads software directly from NVIDIA.

Only official NVIDIA download servers are supported.

Downloaded installers are verified before installation whenever
verification information is available.
