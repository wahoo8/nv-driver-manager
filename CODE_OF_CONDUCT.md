# Code of Conduct

This project has adopted the Contributor Covenant Code of Conduct.

We are committed to providing a welcoming, respectful and harassment-free
environment for everyone.

Expected behavior includes:

- Be respectful.
- Be constructive.
- Assume good intentions.
- Focus on improving the software.

Unacceptable behavior includes:

- Harassment
- Personal attacks
- Discrimination
- Trolling

Project maintainers reserve the right to remove inappropriate comments,
issues or pull requests.

For the full Contributor Covenant 2.1 see:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/
